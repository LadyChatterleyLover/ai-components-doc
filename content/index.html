# 组件

# 趋势组件

# 介绍

趋势符号，标记上升和下降趋势。通常用绿色代表“好”，红色代表“不好”，股票涨跌场景除外

# 用法

<trend>
  <div slot="text">营业额</div>
  5%
</trend>

<trend type="down">
  <div slot="text">营业额</div>
  5%
</trend>

颜色反转

<trend reverseColor>
  <div slot="text">销售量</div>
  5%
</trend>

<trend type="down" reverseColor>
    <div slot="text">成本</div>
    3%
</trend>
参数 说明 类型 可选值 默认值
type 上升或下降 string up / down up
reverseColor 是否反转颜色 boolean true / false false

# 通知图标

# 介绍

通用通知图标,可设置消息条数,自定义消息内容

# 用法

消息为空

<notice-icon></notice-icon>

消息小于100

<notice-icon>5</notice-icon>
<notice-icon>99</notice-icon>

消息大于100

<notice-icon>1500</notice-icon>

通知菜单(用法同element-ui Popover组件)
默认宽度为100,正上方弹出菜单

<notice-icon>
  5
  <template v-slot:content>
    这是一个自定义的菜单
  </template>
</notice-icon>

可以控制菜单的宽度和定位

<notice-icon :width="300" placement="bottom">
  5
  <template v-slot:content>
    可以控制菜单的宽度和定位
  </template>
</notice-icon>
参数 说明 类型 可选值 默认值
width 菜单宽度 string / number - 100
placement 菜单出现位置 string top / top-start / top-end /bottom /bottom-start / bottom-end / left / left-start / left-end / right / right-start /right-end top
v-slot:content 菜单内容插槽 - - -

# 图片预览

# 介绍

通用预览图片,点击图片,图片居中预览,再次点击图片或者空白区域取消预览,父元素可设置图片宽高,如果不设置,则为图片原宽高

# 用法

<image-preview :image="image"></image-preview>

<div style="width: 100px;height: 100px;">
  <image-preview :image="image"></image-preview>
</div>
参数 说明 类型 可选值 默认值
image 图片路径 string 必填 -

PS:目前只支持单图片预览,以后会做轮播图片预览

# 标签选择器

# 介绍

标签选择器,支持单选,全选,反选,视图的展开和收起

# 用法

<tag :tagList="tagList"></tag>

展开和收起

<select-tag :tagList="tagList"></select-tag>
参数 说明 类型 可选值 默认值
tagList 标签组 array 必填 -

PS:tagList需要是一个数组,且数组每个元素为对象,标签名为元素的name属性

# 进度条

# 介绍

在element-ui进度条的基础增加了不显示进度文字,自定义显示内容,进度条加载动画,设置动画加载时间

# 用法

不显示进度

<ai-progress :percentage="66" bg-color="#f46" :noData="true"></ai-progress>

自定义显示内容

<ai-progress :percentage="66" bg-color="#f46" :line-data="true">
  <template v-slot:content>转化率(日同比 66% )</template>
</ai-progress>

进度条加载动画

<ai-progress :percentage="50" bg-color="#f46" :is-animate="true"></ai-progress>

设置加载动画时间

<ai-progress :percentage="50" bg-color="#f46" :is-animate="true" :duration="5000"></ai-progress>
参数 说明 类型 可选值 默认值
percentage 进度 number / string 必填(0-100) -
isAnimate 是否有动画效果 boolean true / false false
duration 动画时长 number / string - 3000
noData 是否不显示数据 boolean true / false false
lineData 是否自定义显示内容 boolean true / false false
v-slot:content 自定义内容插槽 - - -

PS:因为依赖了vue-count-to,所以需要额外安装该插件,如不需要动画效果,则不需要安装

# 全屏

# 介绍

全屏组件,点击进入全屏模式,再次点击或者点击ESC键退出全屏

# 用法

<full-screen></full-screen>

PS:该组件依赖screenfull插件,所以需要额外安装screenfull插件,如果不需要该组件,则不需要

# 底部工具栏

# 介绍

类似谷歌浏览器下载的底部工具栏,内容可自定义

# 用法

<foot-tool-bar>
  <template v-slot:content>
    这是一段内容
  </template>
  <template v-slot:btn>
    <el-button>取消</el-button>
    <el-button type="primary">确定</el-button>
  </template>
</foot-tool-bar>
参数 说明 类型 可选值 默认值
v-slot:content 工具栏内容插槽 - - -
v-slot:btn 右侧帮助按钮插槽 - - -

# 行内可编辑表格

# 介绍

点击编辑,单独一行单元格变成输入框

# 用法

<inline-table :tableData='tableData' :tableDesc='tableDesc'></inline-table>
参数 说明 类型 可选值 默认值
tableData 表格数据 array 必填 -
tableDesc 表格描述 array 必填 -

PS:tableDesc为一个数组,数组的元素为对象,对象只有label和prop属性,对应element-ui表格的label和prop属性

tableDesc 表格描述 array 必填 -

PS:tableDesc为一个数组,数组的元素为对象,对象只有label和prop属性,对应element-ui表格的label和prop属性
